TCC Loucura

Para a bancada desenvolveremos uma interface utilizando o software "Arduino IDE".

Isso por que escolhemos a plataforma de desenvolvimento Arduino Mega para o projeto...

A gente até poderia usar o Uno, mas o Mega fornece mais pinos para utilização.

Optamos por adicionar um Display LCD TFT Touch Shield no projeto, para que tanto a visualização quanto
a interação com a bancada didática fossem facilitadas para o aluno.

E vale ressaltar que devido a limitação do display, não pudemos desenvolver uma interface muito sofisticada
utilizando apenas formas geométricas simples.
